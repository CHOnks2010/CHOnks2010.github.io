What is a DIV element?

An element that groups sections together and will be impacted by the CSS styling. In order to target the div tag, you need to individually identify them.

What is one way you can identify or target a DIV

One way to identify a DIV is to use classes which is a case sensitive tag.

Why would use a DIV element?

To organize your workpage so that users can access sources easily.

What is a border property?

The border property allows you to style the border around an element. The border indicates the size of an element. The border-style property can have up to four values for each of the four sides of the border.

What are some of the different border properties?

Width, Color, Sides, and Rounded.

What is the difference between margins and paddings?

A margin is the spacing between the div and the other element while padding is the spacing between the content and the element.
